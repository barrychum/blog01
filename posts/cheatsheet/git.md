Here are the top 5 sites to learn Git for beginners:

1. **Git Official Documentation** - Comprehensive and official source for all things Git.
   - [Git Documentation](https://git-scm.com/doc)

2. **Codecademy** - Interactive courses on Git basics.
   - [Codecademy Git](https://www.codecademy.com/learn/learn-git)

3. **freeCodeCamp** - Free, interactive tutorials and exercises.
   - [freeCodeCamp Git](https://www.freecodecamp.org/news/what-is-git-learn-git-version-control/)

4. **Atlassian Git Tutorials** - Detailed guides and tutorials from the creators of Bitbucket.
   - [Atlassian Git Tutorials](https://www.atlassian.com/git/tutorials)

5. **Udacity** - Video tutorials and practical exercises.
   - [Udacity Git Course](https://www.udacity.com/course/version-control-with-git--ud123)

For real beginners, the following sites offer excellent tutorials with step-by-step instructions:

1. **Codecademy** - Interactive and beginner-friendly.
   - [Codecademy Git](https://www.codecademy.com/learn/learn-git)

2. **freeCodeCamp** - Free and easy to follow.
   - [freeCodeCamp Git](https://www.freecodecamp.org/news/what-is-git-learn-git-version-control/)

3. **GitHub Learning Lab** - Hands-on, practical exercises within GitHub.
   - [GitHub Learning Lab](https://lab.github.com/)

4. **Git Tutorial by TutorialsPoint** - Simple and clear explanations.
   - [TutorialsPoint Git](https://www.tutorialspoint.com/git/)

5. **Udacity** - Video-based learning with practical exercises.
   - [Udacity Git Course](https://www.udacity.com/course/version-control-with-git--ud123)
